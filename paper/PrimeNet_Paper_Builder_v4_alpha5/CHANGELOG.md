# Changelog

## 4.0.0-alpha.5

### Added

- Figure Engine v1 with typed JSON figure specifications.
- Deterministic PNG, SVG, and JSON metadata outputs.
- Figure catalog generation.
- Bar and line rendering with linear and logarithmic scales.
- Headless Matplotlib `Agg` backend for Windows and CI builds.
- Architecture repository-scale reference figure.
- Figure Engine regression tests.

### Preserved

- Evidence Engine v1.
- Table Engine v1.
- Framework diagnostics, plugin discovery, packaging, and build summaries.
