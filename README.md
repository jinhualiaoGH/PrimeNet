# PrimeNet

PrimeNet is an observational infrastructure for prime arithmetic.

It provides:

- a canonical prime-index coordinate system
- verified prime and gap repositories
- controlled execution and provenance
- reusable scientific instruments
- domain-specific observatories
- reproducible reports and publication materials

## Canonical mathematical objects

For prime index `i`:

```text
p(i) = the i-th prime
g(i) = p(i + 1) - p(i)
