# PrimeNet Architecture

## 1. Infrastructure First

PrimeNet follows an infrastructure-first philosophy.

Before large-scale analysis, PrimeNet builds:

- reproducible repositories,
- manifests,
- verification tools,
- runtime logs,
- metadata,
- indexes,
- and stable APIs.

The goal is to make future arithmetic investigations queryable, reproducible, and scalable.
