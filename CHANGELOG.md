# Changelog

All notable changes to PrimeNet are documented in this file.

PrimeNet follows the principles of reproducible scientific software development. Major milestones correspond to the introduction of new scientific capabilities, observatories, computational instruments, datasets, or public releases.

# [1.0.0] — Foundation of the PrimeNet Observatory

## Repository Infrastructure

* Established the PrimeNet repository structure.
* Introduced standardized directory organization.
* Added documentation framework.
* Added semantic versioning.
* Added licensing and citation metadata.

---

## Prime Repository

* Established the Prime Repository as the permanent archive for raw prime datasets.
* Standardized dataset organization.
* Introduced metadata conventions.
* Defined repository release structure.

# Versioning Policy

PrimeNet follows Semantic Versioning.

* Major versions introduce significant architectural or scientific capabilities.
* Minor versions add new observatories, instruments, atlases, or documented workflows.
* Patch versions improve implementation quality, documentation, reproducibility, or correct defects while preserving scientific compatibility.
